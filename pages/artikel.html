Hypertext Markup Language

HTML (Hypertext Markup Language) adalah bahasa markah yang digunakan untuk membuat dan memformat struktur halaman web. Ini merupakan dasar dari hampir semua situs web di internet. HTML menggunakan elemen-elemen markah, seperti tag dan atribut, untuk mengatur konten di dalam halaman. Hal ini memungkinkan penentuan judul, paragraf, gambar, tautan, daftar, tabel, formulir, video, dan elemen lainnya secara hierarkis. Browser web menguraikan kode HTML dan menghasilkan tampilan visual yang dapat dilihat dan diakses oleh pengguna. HTML seringkali dipadukan dengan CSS (Cascading Style Sheets) untuk mengatur tampilan visual yang lebih menarik dan teratur.
